Predictive Analysis of Breast Cancer Data

Predictive analysis of breast cancer data using data mining involves using statistical and machine learning techniques to analyze large datasets related to breast cancer in order to identify patterns and build predictive models that can help in early detection, diagnosis, and treatment of the disease.

See more

Project Overview

I worked on a Case study on an increasing number of cancer subjects in Wisconsin city for my Data mining final project. I have sampled a clinical trial finding dataset from the research paper published on the open-source ML repository. I identified different error costs and defined a custom metric that would minimize the undesirable error costs. Later, I trained various supervised ML models like Decision trees, Logistic Regression and have them compete against each other and find the best model that would predict a cancer patient accurately and consistently.
