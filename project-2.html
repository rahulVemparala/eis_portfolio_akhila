Music database

An advanced database management system for music databases is designed to efficiently store, manage, and retrieve music-related data, such as artist information, album details, tracklists, and user preferences. The system can also provide advanced search and recommendation features for users based on their music preferences and behavior.

Project Overview

RDBMS concepts are crucial for any IT project since building an efficient data model is the foundation for developing a website or any application. With concepts such as entity relationships and normalization, I have built an efficient data warehousing model using Spotify data. To accomplish this, I used screen/web scraping with an open-source API client from Spotify and loaded the data into an Oracle database to establish relationships. I believe that understanding these concepts will be critical in building a data model, which will be crucial for my website.
